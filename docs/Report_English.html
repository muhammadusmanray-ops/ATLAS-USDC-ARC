3. Machine Learning Framework

3.1 Demand Prediction (Random Forest)

We use a Random Forest Regression model with 10 estimators. The model analyzes historical USDC demand, BTC/ETH volatility, and network congestion to predict the 'Gas-Saving Potential' for the next block.

3.2 Security (Z-Score Guard)

To prevent economic attacks, our Guardian Agents utilize Z-Score statistical analysis. If the incoming traffic demand exceeds a threshold of 2.5 standard deviations, the system automatically triggers a circuit breaker.
